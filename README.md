# Data Store

A lightweight in-memory data store in [Clojure](https://clojure.org/).

### Running unit tests

1. [Install Leiningen](https://leiningen.org/) if you haven't already
2. `lein test`

### Bugs

Please open an [issue](https://github.com/kienstra/data-store/issues) on GitHub.
